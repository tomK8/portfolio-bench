Risk metrics derived from the daily TWR series on the Returns tab. Volatility is the annualised sample stdev of weekday returns × √252 (weekend rows are forward-filled zeros and would compress vol if included). Sharpe = (annualised return − risk-free rate) / annualised vol; Sortino swaps the denominator for downside deviation (excess-return rms over loss days only); Calmar = annualised return / |max drawdown|. Annualised return uses the full daily series compounded; trailing 1y figures use the last 365 days.

Per-symbol GBP P&L over a date window: end_value − start_value + window_cash_flows, where cash flows sum every ledger row for that symbol (buys negative, sells positive, dividends positive, FX baked in at trade date). Captures realised + unrealised gains plus dividends. Trades at fair value contribute zero — only subsequent market moves and dividends do. Bonds use the per-£100 nominal formula. Symbols with no price_history rows are valued at zero on the boundaries (skewing the row). End-value uses live intraday + live FX when the window ends today, daily close + historical FX otherwise. Return % = P&L ÷ peak GBP capital deployed in the window; Annualised % grosses that up to a 365-day year (so a 2% move over a month reads as ~27%).
